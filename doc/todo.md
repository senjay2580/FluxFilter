## 🤖 AI 功能建议

### 高价值 AI 功能

| 功能 | 描述 | 使用场景 |
|------|------|----------|
| **智能摘要** | 一键生成视频/文章的 AI 摘要，节省时间 | RSS 文章、B站视频 |
| **内容分类** | 自动为收藏内容打标签分类 | 收藏夹整理 |
| **相似推荐** | 基于已收藏内容推荐相似优质内容 | 发现新内容 |
| **知识问答** | 基于收藏内容构建个人知识库，支持问答 | 复习学习 |
| **阅读计划** | AI 根据内容长度和你的空闲时间生成阅读计划 | 时间管理 |
| **情绪分析** | 分析收藏内容的情绪倾向，了解自己的内容偏好 | 自我洞察 |

### 推荐优先实现

```
1. 智能摘要 (最实用)
   - 调用 Gemini/GPT API
   - 一键生成 3 句话摘要
   - 可保存到笔记

2. 内容自动分类 (提升体验)
   - 分析标题和描述
   - 自动打标签
   - 智能文件夹

3. 周报生成 (有特色)
   - 统计本周观看/阅读量
   - 生成兴趣热词
   - 给出学习建议
```
